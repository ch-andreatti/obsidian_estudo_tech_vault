# Geral
A definição de qualidade dados utilizada pelo livro **Data Quality Fundamentals**, é:

"Definimos como qualidade de dados, como sendo a saúde dos dados em qualquer parte do seu ciclo."

## Data downtime
Período que os dados ficam indisponíveis para uso, alguns exemplos são:
- Dados faltantes
- Dados inconsistentes (Duplicados, ...)

Podemos usar essa medida, para ver o quão maduro estão os nossos produtos de dados
