**ACID** é um acrônimo que define **quatro propriedades fundamentais** para garantir a confiabilidade em transações com dados (principalmente em bancos de dados e sistemas de arquivos como Lakehouse).

Significado do acrônimo **ACID**:
- A (Atomicidade): A transação ocorre inteira ou não ocorre. Se algo falhar, nada é aplicado;
- C (Consistência): Os dados vão de um estado válido a outro estado válido. Nenhum dado inválido é salvo;
- I (Isolamento): Transações simultâneas não interferem entre si. Como se fossem sequenciais;
- D (Durabilidade): Uma vez confirmada, a transação não será perdida, mesmo com falhas de energia ou sistema.
