# Introdução

Arquitetura de dados é o design de sistemas que irão suprir as demandas por dados levando em consideração flexibilidade ,decisões reversíveis e consideração de trade-offs

Podemos classificar arquitetura em dois tipos:
- Operacional: O que precisa ser feito
- Técnica: Como será feito
