# Introdução
**Design patterns** são templates customizados utilizados para resolverem problemas específicos. A utilização de design patterns agiliza o desenvolvimento e torna as soluções mais robustas, por utilizarem abordagens já testadas e comprovadas

É importante lembrar que **todo design patterns irá trazer uma consequência para o projeto**, como por exemplo em adição de logica extra que aumenta a complexidade, sempre devemos estar cientes das consequências que iremos estar expostos
