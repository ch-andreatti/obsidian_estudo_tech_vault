# Pontos de atenção

Quando uma nova nota for criada, devemos seguir os pontos abaixo para garantir que o novo conteúdo seja aproveitado da melhor maneira possível:
- Conectar o novo conteúdo com os conteúdo já existentes, para fortalecer conexões
- Colocar o contexto no novo conteúdo
	- Por que estou estudando ele? Qual objetivo quero atingir com esse novo conhecimento?
- Fazer a nota com as minhas palavras, para facilitar a absorção do novo conteúdo
